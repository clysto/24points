# 24 Points

Calculate all the solutions of the 24-point game and use the `Reverse Polish notation` to output.

## Usage

```sh
./solve.py 9 1 6 6
[6, 9, 1, '+', 6, '-', '*']
[6, 9, 1, '+', 6, '-', '*']
[6, 1, 9, 6, '-', '+', '*']
[6, 1, 6, 9, '-', '-', '*']
[6, 1, 9, 6, '-', '+', '*']
[6, 1, 6, 9, '-', '-', '*']
[6, 9, 1, 6, '-', '+', '*']
[6, 9, 6, 1, '-', '-', '*']
[6, 9, 1, 6, '-', '+', '*']
[6, 9, 6, 1, '-', '-', '*']
```
